Atividade Avaliativa I

Criar uma página com estilização (arquivo CSS externo).

  1. Passo 1: Criar uma pasta raiz para o projeto "HTML-CSS"
  2. Passo 2: Adicionar um arquivo "index.html"
    • a) Abra a página da sua instituição e procure pelos cursos de extensão disponíveis.
    • b) Adicione um título "Projetos de Extensão 2022".
    • c) Adicione um subtítulo "Tecnologia da Informação"
    • d) Liste pelo menos 2 projetos disponíveis para alunos de tecnologia.
    • e) Para cada projeto coloque uma breve descrição.
    • f) Adicione um "Saiba mais" com o link do site da instituição.
    • g) Adicione uma imagem relacionada ao tema.
    • h) Criar um link para uma página "pagina2.html" dentro do site.
  3. Passo 3: Criar uma pasta Imagens dentro da raiz
    • a) Adicionar as imagens a essa pasta
  4. Passo 4: criar um arquivo style.css na pasta raiz
    • a) Criar os estilos dos elementos da página
  5. Passo 5: Criar uma "pagina2.html"
    • a) Crie uma tabela nesta página e adicione estilização com os seguintes elementos:
    1. i. título com o seu nome;
    2. ii. um pequeno texto com a sua biografia;
  6. b. Fazer com que a página tenha três parágrafos:
    1. i. uma saudação para todos que lerem sua página;
    2. ii. uma descrição física da sua instituição;
    3. iii. uma relação dos nomes dos colegas de grupo.
  7. Utilize lista ordenadas e/ou não ordenadas
  8. d. Introduza no final da página seu endereço de e-mail.
